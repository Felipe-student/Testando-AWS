Sobre o site

Este é um exemplo de um site estático simples, desenvolvido com HTML e CSS. Você pode personalizar o conteúdo conforme necessário.

Como funciona?

Um site estático é composto por arquivos HTML, CSS e, opcionalmente, JavaScript, que são renderizados diretamente pelo navegador. O conteúdo não muda dinamicamente, a menos que haja uma interação com o usuário.
